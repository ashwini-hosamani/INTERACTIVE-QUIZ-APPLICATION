### LuminaQuest - Interactive Quiz Application ✨

**LuminaQuest** is a modern, fully responsive, and interactive quiz web application developed using pure HTML, CSS, and JavaScript. The project aims to provide an engaging platform for users to test their knowledge across various domains while offering a smooth, professional, and visually appealing experience.

## 🛠️ 1. Tools & Technologies Used

The entire application is built using only frontend technologies with no external frameworks or libraries:

- **HTML5**: Used for structuring the semantic and accessible layout of the application, including multiple screens (Home, Difficulty Selection, Quiz, and Results).
- **CSS3**: Responsible for the modern, clean, and professional design. Advanced features like Flexbox, CSS Grid, CSS custom properties (variables), smooth animations, transitions, hover effects, and fully responsive design for mobile, tablet, and desktop were implemented.
- **Vanilla JavaScript (ES6+)**: Handles all the dynamic functionality including category selection, difficulty-based timer system, question loading, answer validation, instant feedback, voice narration, score calculation, and result generation.
- **Web Speech API**: Integrated for the voice-based quiz feature, allowing users to listen to questions by clicking the 🔊 button.
- **LocalStorage**: Used to save the user’s dark mode preference across sessions.
- **Version Control**: Git for local tracking and GitHub for hosting the repository and enabling live deployment.

No backend, databases, or third-party libraries were used, making the project lightweight and easy to understand.

## 💻 2. Editor & Platform Used

- **Code Editor**: Visual Studio Code (VS Code) – chosen for its excellent support for HTML, CSS, and JavaScript, along with useful extensions for live preview and formatting.
- **Operating System**: Windows (Local development on a personal laptop).
- **Primary Browser for Testing**: Google Chrome (best compatibility with Web Speech API and modern JavaScript features).
- **Deployment Platform**: GitHub Pages – a free static hosting service provided by GitHub, allowing the project to be accessed live on the internet without any server setup.
- **Design Inspiration**: Modern minimal UI with attention to typography, spacing, color contrast, and micro-interactions for a delightful user experience.

## 🎯 3. Where This Task Can Be Applicable

LuminaQuest has wide applicability in both educational and professional contexts:

- **Academic Institutions**: Can be used by schools and colleges for conducting subject-wise quizzes, chapter revisions, and competitive exam practice (UPSC, SSC, Banking, GATE, etc.).
- **E-Learning Platforms**: Serves as an excellent interactive module for online learning websites, MOOCs, or self-paced study tools.
- **Corporate Training & HR**: Companies can use it for employee onboarding, skill assessment, domain knowledge testing, and team-building activities.
- **Competitive Exam Preparation**: Particularly useful for aspirants preparing for government exams, aptitude tests, logical reasoning, current affairs, and programming interviews.
- **Personal & Portfolio Projects**: Ideal for students and developers to showcase their frontend development skills, JavaScript proficiency, responsive design capabilities, and ability to handle complex UI logic.
- **Community & Entertainment**: Can be extended for family quiz nights, classroom gamification, social media challenges, or educational YouTube content.

The modular and well-commented code structure makes it easy to extend with new categories, more questions, or additional features like leaderboards and user accounts in the future.

## ✨ Key Features

- 9+ carefully chosen categories including AI, General Knowledge, Aptitude, Programming, Science, Logical Reasoning, UPSC, Current Affairs, and Others.
- Three difficulty levels (Easy – 30 seconds, Medium – 45 seconds, Hard – 60 seconds) with a live countdown timer and visual progress bar.
- Dynamic question loading at runtime with shuffled options for better engagement.
- Instant feedback with color-coded correct and wrong answer highlighting.
- Voice narration support using browser’s built-in speech synthesis.
- Dark mode toggle with saved preference.
- Fully responsive and mobile-friendly design with smooth scrolling and animations.
- Comprehensive result screen showing score percentage, accuracy, and detailed answer review.
- Proper handling of edge cases such as timer expiry, no answer selected, and no questions available.

This project demonstrates strong problem-solving skills, attention to user experience, and clean code practices. It is completely offline-capable and loads quickly even on slower networks.
